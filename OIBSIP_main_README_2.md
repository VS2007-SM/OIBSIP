## 🚀 How to Use

```bash
# Clone the repository
git clone https://github.com/<your-username>/OIBSIP.git
cd OIBSIP

# Task 1: BMI Calculator
cd Task1_BMICalculator
python bmi_calculator.py

# Task 2: Password Generator
cd ../Task2_PasswordGenerator
python password_generator.py

# Task 3: Weather App (run weather_gui.py, not weather_service.py)
cd ../Task3_WeatherApp
python weather_gui.py
```

Each task folder contains its own `README.md` with setup instructions, dependencies (if any), and example output specific to that project.
